CNN Minesweeper Solver

Implemented with PyTorch, the convolutional neural net takes a 5x5 map of mines with the center opened and outputs a 5x5 probability map representing the probability of every block being a mine. The program then opens blocks with a non-mine probability greater than 90%.

This solver is able to achieve high accuracy. Solving a 16x16 map with 40 mines, which performs well on near-center locations, however, often fails to completely finish the game when encountering edges.

The entire environment is implemented with PyGame in OpenAI Gym style. With the specified dicrete action-observation spaces and reward given for each action, the environment makes attempts for other ML/RL algorithms to solve this problem easier.
